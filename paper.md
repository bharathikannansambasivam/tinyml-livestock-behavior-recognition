# Summary

This software implements a TinyML-based livestock behavior recognition system using a lightweight Convolutional Neural Network (CNN) deployed on edge devices. The system analyzes images captured from a camera and classifies livestock behaviors such as grazing, fence proximity, and no-animal detection.

# Statement of Need

Livestock monitoring is essential for maintaining animal health and improving farm productivity. Traditional monitoring methods rely on manual observation, which is inefficient for large farms. This software provides a lightweight machine learning solution that performs real-time livestock behavior recognition on edge devices using TensorFlow Lite.

# Software Description

The system uses computer vision techniques and a trained CNN model to detect livestock behavior. The model is converted into TensorFlow Lite format to enable efficient inference on resource-constrained devices.
