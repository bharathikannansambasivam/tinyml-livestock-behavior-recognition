# Livestock Behavior Recognition using TinyML

## 📌 Project Overview

This project presents a lightweight livestock behavior monitoring system developed using Computer Vision and Tiny Machine Learning (TinyML). The system is designed to detect and classify livestock behaviors such as grazing, fence proximity, and no-animal conditions in real time using a live camera feed.

The main objective of this project is to provide a low-cost, edge-based monitoring solution that works without cloud dependency or additional hardware sensors.

---

## 🎯 Features

- Real-time livestock behavior detection
- Camera-based monitoring system
- Lightweight CNN model
- TensorFlow Lite (TFLite) deployment
- Offline processing (No internet required)
- Alert generation for risky behavior (Fence proximity)

---

## 🧠 Technologies Used

- Python
- TensorFlow
- TensorFlow Lite
- OpenCV
- NumPy

---

## 🏗️ System Architecture

Camera → Preprocessing → TinyML CNN Model → Behavior Classification → Alert System

---

## 📊 Behavior Classes

- Grazing
- Fence Proximity
- No Animal
